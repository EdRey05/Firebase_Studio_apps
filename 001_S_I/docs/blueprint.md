# **App Name**: FinTrack Horizon

## Core Features:

- Account Management: Create and manage multiple savings and investment accounts with details like name, bank, interest rate, and creation date. Ability to delete accounts.
- Log Entry Management: Log entries for contributions, withdrawals, and interest gained with dates. Includes interactive widgets to add these entries to each account.
- Growth Visualization: Display interactive plots showing the gradual increase of money over time, with month labels for easy visualization. Users can adjust the range shown on the x-axis (months or years).
- Transaction Log: Show a full log of transactions as a scrollable table, with the ability to add new entries for the selected account directly from the dashboard.
- Projected Growth Modeling: Project future account growth based on current balance, user-defined interest rate, and monthly contributions. Adjustable widgets for interest rate, monthly contributions, and projection time frame.
- AI Savings Suggestion: Generative AI tool suggests potential savings adjustments to achieve desired future values based on user inputted goals, current trajectory, and risk tolerance.
- Data Persistence: The app will read and save the data into a local csv file.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for a sense of stability and growth.
- Background color: Light gray (#F0F4F7) for a clean, modern look.
- Accent color: Soft green (#A5D6A7) to highlight gains and positive actions.
- Body and headline font: 'Inter' sans-serif, for a modern and objective look. Suited for both headlines and body text.
- Use clear, minimalistic icons to represent different account types and actions.
- Dashboard layout will feature a clear, tabbed interface. Key data visualizations prominently displayed. Scrollable table clearly presents full log.
- Subtle animations on data updates and transitions to provide a smooth user experience.
